# Universal Shop Bot

Универсальный Telegram-бот-магазин для продажи товаров, услуг, цифровых продуктов и подписок.

## Что умеет первая версия

- запуск Telegram-бота;
- главное меню покупателя;
- каталог товаров;
- админ-меню;
- подготовленная структура для добавления, удаления и редактирования товаров;
- подготовленная структура базы данных;
- настройки через `.env`.

## Технологии

- Python 3.11+
- aiogram 3
- SQLite
- python-dotenv

## Структура проекта

```text
bot/                 запуск бота
config/              настройки проекта
database/            база данных и схема
handlers/            обработчики команд и кнопок
keyboards/           клавиатуры Telegram
models/              модели данных
services/            бизнес-логика
media/               изображения товаров
docs/                документация
```

## Быстрый старт

1. Создать виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate
```

Для Windows:

```bash
.venv\Scripts\activate
```

2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Создать `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

4. Заполнить `.env`:

```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_IDS=123456789
DATABASE_PATH=shop.db
```

5. Запустить бота:

```bash
python main.py
```
